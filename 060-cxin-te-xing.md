#### C++ 新特性


#### auto
**1 可以从初始化表达式中推断出变量的类型, 大大简化编程工作**
**2 属于编译器特性, 不影响最终的机器码质量, 不影响运行效率**
```
// 以前我们定义变量, 是这样的 (指定变量的类型)
int a = 10;
double d = 20.0
Person *p = new Person(10);
p->m_age = 30;

// 新编译器使用auto, 编译器自动推到出类型
auto a = 10;
auto d = 20.0
auto *p = new Person(10);
p->m_age = 30;
```
